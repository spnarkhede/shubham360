# PromptVault

> *The right prompt is half the solution.*

PromptVault is a personal, living library of prompts — built from real work, real experiments, and real results. Each section maps to a specific AI agent and the kinds of tasks that agent handles best.

---

## How It's Organized

Each agent gets its own folder. Inside each folder, prompts are grouped by **what you're trying to do** — not by the agent's feature list.

| Agent | Best For |
|---|---|
| [ChatGPT](./ChatGPT/) | Writing, ideation, analysis, research |
| [Claude](./Claude/) | Deep reasoning, long docs, code review |
| [GitHub Copilot](./Copilot/) | In-editor coding, autocomplete, refactoring |
| [DeepSeek](./DeepSeek/) | Code generation, algorithmic thinking |
| [Gemini](./Gemini/) | Google Workspace, multimodal, real-time |
| [Grok](./Grok/) | Real-time web context, X/Twitter data |
| [Midjourney](./Midjourney/) | Image generation, visual design |
| [VideoAI](./VideoAI/) | Video scripts, AI video generation |
| [Nano Banana](./NanoBanana/) | Local/lightweight AI, edge use cases |

---

## How to Use This

1. **Pick your agent** from the sidebar
2. **Browse by category** — each folder has prompt files by use case
3. **Copy, adapt, run** — prompts are written to be used, not admired
4. **Add your own** — this is a living document

---

## Prompt Format Used Here

Every prompt entry follows this structure:

```
## Prompt Title

**Use case:** What problem this solves  
**Agent:** Which AI this works best with  
**Tone:** Formal / Casual / Technical

---

[The actual prompt text, ready to copy]

---

**Tips:** What to tweak for better results
```
