## [1.0.0] — 2026-02-19

### Initial Release

**Templates**
- `templates/MEMORY.md` — Living AI context and architectural decisions template
- `templates/COGNITIVE_DEBT_CHECKLIST.md` — Developer pre-accept checkpoint card
- `templates/TEAM_PLAYBOOK.md` — Sprint ceremonies and quarterly audit system

**Guidelines**
- `AI_USAGE_GUIDELINES.md` — Team policy: permitted use, prohibited behaviors, roles
- `CODE_REVIEW_GUARDRAILS.md` — Five-layer review framework for AI-generated code
- `DEFINITION_OF_DONE.md` — Tiered DoD (Standard / Elevated / High-Risk) for AI-assisted tickets

**Procedures**
- `AI_TOOL_ONBOARDING.md` — 90-minute structured onboarding before first AI-assisted PR
- `INCIDENT_RESPONSE_PROCEDURE.md` — Five-phase response for AI-related production incidents
- `ESCALATION_PROCEDURE.md` — When, how, and to whom to raise cognitive debt concerns

**GitHub**
- `PULL_REQUEST_TEMPLATE.md` — PR comprehension gate with mandatory AI declaration
- `ISSUE_TEMPLATE/improvement.md` — Structured improvement proposals
- `ISSUE_TEMPLATE/bug_report.md` — Bug and inconsistency reports

**Grounded in:**
- Margaret-Anne Storey's cognitive debt research (Feb 2026)
- METR randomized controlled trial on AI-assisted coding (Jul 2025)
- Stack Overflow 2025 Developer Survey
- Cortex 2026 Engineering Benchmark
- DORA 2024–2025 Accelerate State of DevOps
- GitClear code churn analysis
- Steve Yegge's "The AI Vampire" (Feb 2026)
