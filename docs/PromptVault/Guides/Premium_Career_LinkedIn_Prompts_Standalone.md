## How to Use This Library

1. Copy one prompt per AI chat session. One section per session keeps output quality high.
2. Replace every `[BRACKET]` before running. The 7-line context block at the top of each prompt does the heavy lifting; fill it with real numbers.

4. Re-run the audit prompts (3, 9, 18, 23, 24) every 60 days as your profile and the market move.
5. Reality note: no prompt guarantees a response rate or a placement. These prompts maximize what you control: search ranking, message quality, and proof presentation.
