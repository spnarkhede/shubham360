> Generate stunning digital illustrations with this AI prompt, featuring dramatic lighting, high-contrast styles, dynamic compositions, and customizable character elements.

### Prompt

```
{
  "art_metadata": {
    "medium": "Digital illustration",
    "style": ["Striking", "High-contrast"]
  },
  "subject": {
    "figure": "[CHARACTER]",
    "attire": "Red hooded jacket",
    "pose": "Head tilted upwards",
    "perspective": "Low angle view"
  },
  "scene_elements": {
    "dynamic_action": {
      "description": "Chaotic swirl of [SWARM ELEMENT] taking flight",
      "interaction": "Merging into a vibrant, intense red and orange glow"
    },
    "foreground": "Lower part shrouded in dark, smoky shadows"
  },
  "lighting_and_atmosphere": {
    "type": "Dramatic",
    "source": "Powerful red backlight",
    "effects": "Illuminating [SWARM ELEMENT] and surrounding atmosphere",
    "palette": ["Intense Red", "Vibrant Orange", "Black", "Smoky Dark Grey"]
  }
}
```

### What it does

- Creates a dramatic AI image of a character in a red hooded jacket looking upward from a low angle view.
- Generates a chaotic swirl of elements taking flight that merge into intense red and orange glowing effects.
- Uses powerful red backlighting against dark smoky shadows to create high-contrast striking atmosphere.

### Tips

- Choose a character that fits the dramatic mood, like a mysterious wanderer or urban explorer, and select a swarm element such as birds, butterflies, or leaves to create the chaotic flight effect.
- Consider swarm elements that naturally complement your color palette, using creatures like fireflies for warm glows or ravens for darker contrast against the red backlight.
- Use the JSON prompt to guide the AI in creating a high-contrast digital illustration with dynamic movement and atmospheric lighting.
