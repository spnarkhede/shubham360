### 5. Technical Skills Section Builder

**Use Case:** Build a strong technical skills section from a short experience summary.

**Target Personas:** Job Seeker, Individual Contributor

**Tags:** `Job Seeker`, `Resume Writing`, `Skills Section`

**Prompt:**

```
I want to include a section on technical skills and tools, but I'm not sure what to list or how to format it. Based on my experience [brief description], can you help me write this section in a way that stands out to hiring managers?
```
