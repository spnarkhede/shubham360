> Generate stunning surreal portrait photography with this AI prompt, featuring miniature clones, fashion styling, studio lighting, and photorealistic detail for creative campaigns.

### Prompt

```
{
  "prompt": "An ultra-realistic, full-body studio portrait of a [SUBJECT DESCRIPTION]. She has dark hair styled exactly as in the reference image and wears natural makeup with a serious, direct gaze into the camera. She is wearing [OUTFIT DESCRIPTION], and white Puma athletic shoes.",
  "style_and_pose": "She is seated gracefully on a pristine white platform.",
  "creative_concept": "Multiple miniature versions of herself in the identical outfit are interacting with her: one perched on her head, two sitting on her shoulders, one seated in her open palm, one climbing her leg, and three walking nearby on the platform. All miniature figures must have the same serious expression and perfect likeness.",
  "lighting_and_environment": "Seamless white studio background, high-end fashion lighting, ultra-detailed, photorealistic.",
  "constraints": "Facial likeness must be 100% accurate to the provided reference image. Hair style must be 100% accurate to the provided reference image.",
  "signature": "[NAME]"
}
```

### What it does

- Generates an ultra-realistic AI image of a woman surrounded by multiple miniature versions of herself in identical outfits.
- Creates a high-end studio portrait where tiny clones interact with the main subject by sitting on shoulders, climbing, and walking on a white platform.
- Ensures perfect facial likeness and hairstyle accuracy while maintaining a serious expression across all figures in the AI image prompt.

### Tips

- Choose a subject description that highlights unique physical features and personality traits to help the AI image create an accurate full-body portrait.
- Select an outfit description that complements the creative concept with the miniature figures, considering textures and colors that photograph well in studio lighting.
- Provide a high-resolution reference image showing clear facial features and exact hair styling to ensure the AI image prompt achieves 100% accurate likeness across all miniature versions.
