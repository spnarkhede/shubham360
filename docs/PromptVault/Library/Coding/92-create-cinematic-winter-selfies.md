> Generate stunning winter selfie portraits with this AI prompt, featuring realistic snow effects, natural lighting, cinematic atmosphere, and detailed facial features.

### Prompt

```
{
  "pose": {
    "description": "She holds the camera with her left hand, positioned slightly below eye level, capturing her face from a confident upward angle; lips softly parted, expression calm yet intense."
  },
  "background": {
    "description": "A dark winter night sky with faint, lightly frosted tree branches barely visible, adding a dramatic atmosphere."
  },
  "atmosphere": {
    "description": "Cold, cinematic, and raw winter energy; a stylish modern snowy portrait with natural texture and detail."
  },
  "reference_style": {
    "description": "Conceptual variant.",
    "details": {
      "shot": "A close-up outdoor selfie taken at night during heavy snowfall.",
      "subject": "The [SUBJECT]'s facial features match the reference image exactly.",
      "skin": {
        "description": "Natural tone with visible, realistic skin texture. Her cheeks are slightly flushed from the cold."
      },
      "snow_details": {
        "description": "Snowflakes rest on her lashes and hair."
      },
      "outfit": {
        "description": "She wears a [HEADWEAR] almost fully covered in snow, and a [OUTERWEAR] with snow dusted across the shoulders and collar."
      },
      "camera": {
        "description": "She is taking the selfie with her left hand."
      },
      "lighting": {
        "description": "Soft, cold night lighting from above and slightly in front, creating a subtle glow on her cheeks and gentle highlights on the jacket."
      }
    }
  },
  "negative_prompt": {
    "description": [
      "blurry face",
      "altered facial features",
      "incorrect facial structure",
      "distorted proportions",
      "extra limbs",
      "extra fingers",
      "missing fingers",
      "deformed eyes",
      "inaccurate expression",
      "unrealistic skin texture",
      "plastic skin",
      "oversoftened skin",
      "unnatural lighting",
      "harsh shadows",
      "artifacts",
      "pixelation",
      "low resolution"
    ]
  }
}
```

### What it does

- Generates a cinematic winter selfie AI image showing a subject holding a camera in cold, snowy night conditions.
- Creates realistic details like snowflakes on lashes and hair, flushed cheeks, and natural skin texture under soft night lighting.
- Captures an upward-angle close-up portrait with dramatic dark sky background and snow-covered clothing for authentic winter atmosphere.

### Tips

- Choose a specific subject identity that matches your creative vision, such as a fashion model, influencer persona, or character concept to fill the SUBJECT variable in this AI image prompt.
- Select winter outfit details that complement the snowy atmosphere, like a wool beanie or fur-lined hood for HEADWEAR and a puffer jacket or vintage parka for OUTERWEAR to create authentic cold-weather styling.
- Use the JSON prompt to guide the AI in creating a cinematic winter selfie scene with natural skin texture and realistic snowfall details.
