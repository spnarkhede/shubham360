### 5. Industry-Specific Cover Letter

**Use Case:** Tailor a cover letter to the expectations and language of a specific industry.

**Target Personas:** Job Seeker, Software Engineer, Healthcare Professional

**Tags:** `Job Seeker`, `Writing`, `Cover Letter`

**Prompt:**

```
Write two industry-specific cover letter variants for me:
1) A software engineering version that emphasizes coding skills and project experience,
2) A healthcare version that emphasizes patient care, teamwork, and reliability.

Resume: [PASTE]
Target role: [PASTE]
```
