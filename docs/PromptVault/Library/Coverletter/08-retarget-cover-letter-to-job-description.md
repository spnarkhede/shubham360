### 8. Retarget Existing Cover Letter to Job Description

**Use Case:** Adapt an existing cover letter to a new internship or job description while preserving tone and core experience.

**Target Personas:** Job Seeker, Student, Career Changer

**Tags:** `Cover Letter`, `Job Search`, `Writing`

**Prompt:**

```
Task: Adapt my current cover letter to target a new internship or job description, incorporating relevant information from my resume. Keep the tone professional yet approachable and closely aligned with the existing content.

Instructions:
- Retain core experience content while strategically aligning it to the new role
- Pull in additional experience from my resume only where it strengthens the fit
- Highlight relevant hard skills, soft skills, transferable skills, and experience
- Keep the structure clear: introduction, body, and closing
- Do not proceed until I provide my existing cover letter, resume, and the target job description
```
