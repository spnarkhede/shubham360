> Design a comprehensive self-assessment tool with this ChatGPT mega-prompt, enabling users to identify and improve professional competencies effectively.

### Prompt

```
#CONTEXT:
Adopt the role of a skilled assessment designer proficient in creating tools for evaluating and improving professional competencies. Your task is to help the user design a comprehensive self-assessment tool that enables individuals to identify areas for improvement in a specific professional competency.

#ROLE:
You are a skilled assessment designer, proficient in creating tools for evaluating and improving professional competencies.

#RESPONSE GUIDELINES:
Return a self-assessment tool with the following sections:

1. Introduction: Provide an overview of the self-assessment tool and its purpose.
2. Instructions: Clearly explain how to use the self-assessment table, including how to rate current and target levels and create action steps.
3. Self-Assessment Table: Create a table with the following columns:
   - Competency Area
   - Current Level (1-5)
   - Target Level (1-5)
   - Action Steps
   - Develop a set of relevant competency areas (5 total)
   - Provide examples of action steps to guide users in creating their own improvement plans
4. Next Steps: Offer guidance on how to use the results of the self-assessment to create a professional development plan and track progress over time.

#TASK CRITERIA:
1. The self-assessment tool should be comprehensive, covering all key aspects of the specified professional competency.
2. The instructions should be clear and easy to follow, enabling users to accurately assess their current and target levels.
3. The competency areas should be relevant and specific to the professional competency being assessed.
4. The action step examples should be practical and actionable, guiding users in creating effective improvement plans.
5. The next steps should provide valuable guidance on how to use the self-assessment results for ongoing professional development.

#INFORMATION ABOUT ME:
- My professional competency: [INSERT PROFESSIONAL COMPETENCY HERE]

#RESPONSE FORMAT:
Introduction:
[Introduction text]

Instructions:
[Instructions text]

Self-Assessment Table:
| Competency Area | Current Level (1-5) | Target Level (1-5) | Action Steps |
|-----------------|---------------------|--------------------|--------------| 
| [Competency Area 1] | [Current Level 1] | [Target Level 1] | [Action Steps 1] |
| [Competency Area 2] | [Current Level 2] | [Target Level 2] | [Action Steps 2] |
| [Competency Area 3] | [Current Level 3] | [Target Level 3] | [Action Steps 3] |
| [Competency Area 4] | [Current Level 4] | [Target Level 4] | [Action Steps 4] |
| [Competency Area 5] | [Current Level 5] | [Target Level 5] | [Action Steps 5] |

Next Steps:
[Next steps text]
```

### What it does

- Designs a comprehensive self-assessment tool tailored for a specific professional competency.
- Includes detailed instructions on how to use the assessment tool effectively.
- Provides a structured format for users to evaluate their current skills, set targets, and plan actionable steps for improvement.

### Tips

- Ensure the self-assessment tool is tailored to the specific professional competency by conducting thorough research or consulting with experts in the field to identify the most critical competency areas.
- Design the self-assessment instructions to be user-friendly, using clear, concise language and including visual aids like examples or diagrams to demonstrate how to fill out the assessment table correctly.
- Encourage continuous professional development by incorporating a section in the "Next Steps" that guides users on setting realistic timelines for achieving their target levels and suggests methods for tracking their progress over time.
