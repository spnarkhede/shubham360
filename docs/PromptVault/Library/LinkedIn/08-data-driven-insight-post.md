### 8. Data-Driven Insight Post

**Use Case:** Convert one statistic into a practical insight post that builds authority.

**Target Personas:** Analyst, Manager, Individual Contributor

**Tags:** `LinkedIn`, `Analysis`, `Thought Leadership`

**Prompt:**

```
Using this data/statistic: [insert], write a LinkedIn post (<=220 words) that explains why it matters in [industry], connects it to an emerging skill, and shares one practical example.
```
