### 22. Live Session Promotion Post

**Use Case:** Promote a webinar or live session with clearer value and stronger sign-up intent.

**Target Personas:** Creator, Educator, Consultant

**Tags:** `LinkedIn`, `Events`, `Promotion`

**Prompt:**

```
Draft a LinkedIn post to promote an upcoming live session or webinar.
Topic: [topic]
Date and time: [date/time]
Target audience: [audience]

Include an attention-grabbing headline, 3 bullet points on what attendees will learn, one personal anecdote about why I am hosting it, and a CTA to register or set a reminder.
```
