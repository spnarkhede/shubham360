### 25. Live Q and A Recap and Slide Deck

**Use Case:** Repurpose a live session into a recap deck and a follow-up LinkedIn post.

**Target Personas:** Creator, Educator, Consultant

**Tags:** `LinkedIn`, `Content Repurposing`, `Events`

**Prompt:**

```
For a live session on [topic], write a 5-slide recap deck outline covering key questions asked, my answers, the biggest takeaway, and the next step. Then draft a 120-word LinkedIn post highlighting the biggest insight and inviting viewers to rewatch or join the next session.
```
