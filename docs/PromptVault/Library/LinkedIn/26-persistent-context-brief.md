### 26. Persistent Context Brief

**Use Case:** Store resume and profile context once so later LinkedIn tasks can reuse it without repeated uploads.

**Target Personas:** Job Seeker, Consultant, Individual Contributor

**Tags:** `LinkedIn`, `Context Management`, `Profile Optimization`

**Prompt:**

```
I have attached my resume, LinkedIn PDF, cover letter, and any relevant job descriptions or target roles. Treat all of this as persistent context for the conversation.

Read, extract, and structure the important details so I do not have to repeat them. Reuse this context for LinkedIn improvements, job matching, outreach, and interview prep.

Start with a concise analysis of strengths, gaps, and alignment to [ROLES].
```
