### 1. Headline Master Prompt

**Use Case:** Build a formula-based headline from CV points, target roles, and strongest hooks.

**Target Personas:** Job Seeker, Professional, Personal Brand Builder

**Tags:** `LinkedIn`, `Headline Writing`, `Profile Optimization`

**Prompt:**

```
Draft a highly optimized LinkedIn headline from my CV points using this formula:

Current Role | Critical Skill 1 | Critical Skill 2 | Hook 1 - Big Achievement | Hook 2 - Big Brand | Hook 3 - Awards or Certifications

Full prompt:
Draft the critical skills for: [insert target roles]
Get the hooks from: [insert key CV points]
```

### 2. About Section Master Prompt

**Use Case:** Create an About section that balances professionalism and personality with a proven structure.

**Target Personas:** Job Seeker, Professional, Personal Brand Builder

**Tags:** `LinkedIn`, `About Section`, `Profile Optimization`

**Prompt:**

```
Write a LinkedIn About section that balances professionalism and personality.

Follow this structure:
- Strong Hook
- Mission: what drives my career
- Key Experiences: 1 to 3 career highlights
- Core Competencies: keywords plus impact
- Strong Closing: forward-looking statement

Keep it concise, engaging, and recruiter-friendly using this experience: [insert work experience].
```

### 3. Work Experience Master Prompt

**Use Case:** Rewrite LinkedIn work history with recruiter-friendly structure and outcomes.

**Target Personas:** Job Seeker, Professional, Career Changer

**Tags:** `LinkedIn`, `Experience Section`, `Achievement Writing`

**Prompt:**

```
Craft an impactful Work Experience section for LinkedIn for [industry] recruiters.

Format:
- One-line summary: role plus overall impact
- Key achievements: 3 to 5 bullets
- Use the pattern: What + How + Result
- Start with action verbs
- Quantify everything
- Use industry keywords
- Tailor for target designations: [insert roles]
```

### 4. Keywords Master Prompt

**Use Case:** Surface the most important keywords that improve discoverability for target roles.

**Target Personas:** Job Seeker, Professional, Career Changer

**Tags:** `LinkedIn`, `Keywords`, `Search Visibility`

**Prompt:**

```
Identify the most important keywords for my target LinkedIn profile based on [industry], [target roles], and my CV points. Then explain where each keyword should appear across my headline, About section, experience, skills, and featured content so my profile becomes more discoverable.
```
