> Generate stunning fashion editorial portraits with this AI prompt, featuring customizable styling, accessories, lighting, and photorealistic summer aesthetics for creative projects.

### Prompt

```
{
  "subject": {
    "description": "A young woman in an HD editorial fashion portrait",
    "age": "young adult",
    "expression": "soft, warm, intimate expression",
    "hair": {
      "color": "black",
      "style": "long layered cut with loose wind-blown strands and dark bangs under the headscarf"
    },
    "clothing": {
      "top": {
        "type": "[FASHION AESTHETIC] oversized cardigan",
        "color": "[FASHION AESTHETIC color palette]",
        "details": "textured, cozy, fashion editorial style"
      },
      "headscarf": {
        "type": "[FASHION AESTHETIC] headscarf tied under the chin",
        "color": "[FASHION AESTHETIC color palette]",
        "details": "bright, patterned, summery"
      }
    },
    "face": {
      "preserve_original": true,
      "makeup": {
        "cheeks": "soft blush on cheeks and nose",
        "lips": "pink lips",
        "eyebrows": "soft natural eyebrows",
        "overall": "subtle and identical to reference"
      }
    }
  },
  "accessories": {
    "sunglasses": {
      "type": "large square oversized sunglasses",
      "frame": "thick tortoise-shell",
      "lens": "amber-tinted",
      "style": "high-fashion"
    },
    "animal": {
      "type": "[ANIMAL COMPANION]",
      "description": "[ANIMAL COMPANION description] relaxing on her shoulder"
    }
  },
  "photography": {
    "camera_style": "editorial fashion photography",
    "lighting": "intense backlit sunlight with flares, dreamy high-contrast summer aesthetic",
    "angle": "close-up portrait",
    "shot_type": "intimate, warm, photorealistic"
  },
  "background": {
    "wall_color": "bright blue sky",
    "elements": [],
    "atmosphere": "vibrant, summery, high-contrast",
    "lighting": "strong natural sunlight"
  }
}
```

### What it does

- Generates an AI image of a young woman in editorial fashion style wearing an oversized cardigan, patterned headscarf, and large tortoise-shell sunglasses.
- Creates a warm summer portrait with intense backlit sunlight, lens flares, and high-contrast lighting against a bright blue sky.
- Features an animal companion resting on her shoulder while preserving natural makeup details and wind-blown layered hair.

### Tips

- Choose a fashion aesthetic that defines your AI image prompt style, such as bohemian for earthy tones and flowing fabrics, vintage for retro patterns and muted colors, or modern minimalist for clean lines and neutral palettes.
- Select an animal companion that complements the editorial fashion vibe, like a colorful parrot for tropical energy, a sleek cat for sophisticated elegance, or a small bird for delicate charm.
- Use the JSON prompt to guide the AI in creating a cohesive color story where the cardigan and headscarf share the same aesthetic palette, ensuring your fashion editorial AI image maintains visual harmony.
