> Design sustainable urban neighborhoods with this AI prompt, featuring walkable streets, green spaces, mixed-use buildings, and human-centered public areas.

### Prompt

```
{
  "project_type": "urban_design_architecture",
  "concept": "human-centered, sustainable city life",
  "urban_scale": {
    "scope": "neighborhood to city scale",
    "focus": [
      "public spaces",
      "streets and plazas",
      "green infrastructure",
      "mixed-use environments"
    ]
  },
  "design_principles": {
    "human_centered": true,
    "walkability": true,
    "accessibility": true,
    "social_interaction": true,
    "sustainability": true
  },
  "architecture_elements": {
    "building_types": [
      "residential blocks",
      "office buildings",
      "cultural centers",
      "cafes and retail"
    ],
    "scale": "mid-rise, human-scale buildings",
    "material_language": [
      "[URBAN MATERIAL PALETTE]"
    ]
  },
  "public_space_design": {
    "elements": [
      "[PUBLIC SPACE CHARACTER]"
    ],
    "landscape": [
      "[PUBLIC SPACE CHARACTER]"
    ]
  },
  "urban_context": {
    "climate_region": "[CLIMATE REGION]",
    "cultural_reference": "[CLIMATE REGION]"
  },
  "people_and_activity": {
    "users": [
      "pedestrians",
      "cyclists",
      "families",
      "outdoor cafe patrons",
      "children playing"
    ],
    "density": "lively but not overcrowded",
    "scale": "realistic human proportions"
  }
}
```

### What it does

- Generates an AI image of a sustainable urban neighborhood with mixed-use buildings and green public spaces.
- Creates a human-scale city environment where pedestrians, cyclists, and families interact in walkable streets and plazas.
- Produces a lively architectural scene featuring mid-rise buildings, cultural centers, and outdoor cafes with realistic human activity.

### Tips

- Select an urban material palette that reflects your climate region, such as light stone and timber for Mediterranean areas or brick and steel for temperate zones.
- Define your public space character with specific landscape elements like tree-lined promenades for European contexts or native gardens for tropical climates.
- Use the JSON prompt to guide the AI in creating a human-centered urban design that balances your chosen climate region with cultural references for authentic city life scenes.
