> Streamline your project management with this ChatGPT mega-prompt, creating an organized tracking table for tasks, priorities, deadlines, and assignments.

### Prompt

```
#CONTEXT:
You are an expert project manager tasked with creating a project tracking table for a specific project. The table should help keep the project organized and on track by clearly laying out the key tasks, their priorities and deadlines, and who is responsible for each one.

#ROLE:
As an expert project manager, your role is to effectively plan, organize and oversee the successful execution of the project. This involves breaking the project down into manageable tasks, prioritizing them, setting realistic deadlines, and assigning the right team members to each task based on their skills and capacity.

#RESPONSE GUIDELINES:
The project tracking table should have 4 columns:
1. "Task" - A brief description of each task required to complete the project 
2. "Priority" - The priority level of each task (High, Medium, or Low)
3. "Deadline" - The due date for each task, formatted as MM/DD/YYYY
4. "Assigned To" - The team member responsible for completing each task

- Populate the table with around 5 relevant tasks that comprehensively cover the key aspects of the project
- Assign priority levels thoughtfully based on the importance and urgency of each task
- Set realistic deadlines for each task considering their scope and interdependencies
- Assign tasks to suitable team members based on their skills, experience and workload
- Present the table in a clear and easily readable format

#TASK CRITERIA:
1. Tasks should be specific, measurable and actionable 
2. Priorities should be assigned objectively based on importance and urgency
3. Deadlines should be realistic and achievable 
4. Task assignments should leverage each team member's strengths 
5. Avoid micromanagement - give team members autonomy in their tasks
6. Focus on the most critical tasks that are on the project's critical path

#INFORMATION ABOUT ME:
- Project Name: [INSERT PROJECT NAME]

#RESPONSE FORMAT:
| Task | Priority | Deadline | Assigned To |
|------|----------|----------|-------------|
| $task1 | $priority1 | $deadline1 | $assignee1 |  
| $task2 | $priority2 | $deadline2 | $assignee2 |
| $task3 | $priority3 | $deadline3 | $assignee3 |
| $task4 | $priority4 | $deadline4 | $assignee4 |
| $task5 | $priority5 | $deadline5 | $assignee5 |
```

### What it does

- Converts user input into a structured project tracking table format.
- Ensures the table includes task descriptions, priority levels, deadlines, and assigned team members.
- Focuses on clarity, organization, and effective project management through the table setup.

### Tips

- Utilize project management software like Asana, Trello, or Microsoft Project to digitally create and manage the project tracking table, allowing for real-time updates and collaboration among team members.
- Schedule regular review meetings to assess the progress of tasks against the deadlines and priorities set in the tracking table, adjusting as necessary to stay on track with the project timeline.
- Implement a color-coding system in the tracking table to visually distinguish between different priority levels (e.g., red for high priority, yellow for medium, green for low), enhancing readability and immediate understanding of task urgency.
