> Develop a structured checklist for migrating to Azure with a focus on cost optimization, tailored for advanced users with a foundational understanding of cloud computing principles

### Prompt

```
Develop a structured checklist for migrating to Azure with a focus on cost optimization, tailored for advanced users with a foundational understanding of cloud computing principles. The checklist should cover the entire migration process, from initial assessment to post-migration monitoring, and include actionable strategies for cost optimization.** Ensure the checklist addresses the following key areas:** ## 1. **Pre-Migration Assessment**: * Evaluate current infrastructure and applications for Azure compatibility. * Identify potential cost savings through right-sizing and resource optimization. ## 2. **Migration Planning**: * Determine the most suitable Azure services for each workload. * Develop a migration timeline and resource allocation plan. ## 3. **Migration Execution**: * Execute the migration plan, leveraging Azure tools and best practices. * Implement cost optimization strategies, such as reserved instances and autoscaling. ## 4. **Post-Migration Monitoring and Optimization**: * Configure monitoring and logging to track resource utilization and costs. * Continuously evaluate and refine cost optimization strategies based on usage patterns. ## 5. **Cost Management and Governance**: * Establish cost allocation and chargeback processes. * Implement governance policies for resource provisioning and usage. Include technical insights and real-world examples to illustrate key points and facilitate a seamless and cost-effective transition to Azure.
```

### What it does

- Pre-Migration Assessment
- Post-Migration Monitoring and Optimization
- Cost Management and Governance

### Tips

- Specify your cloud provider (AWS / Azure / GCP) at the start for provider-specific guidance.
- Mention regulatory requirements (GDPR, HIPAA) so the response can address compliance directly.
- Ask for a cost estimate breakdown when evaluating architecture decisions.

### How to use

1. Open ChatGPT or Claude and paste the prompt.
2. Provide details about your current infrastructure, cloud provider preference, and compliance needs.
3. Work through the framework section by section, requesting elaboration on complex areas.
4. Use the output as the basis for a technical proposal or architecture review document.
