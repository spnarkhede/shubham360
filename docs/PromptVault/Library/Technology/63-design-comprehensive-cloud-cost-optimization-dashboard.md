> Design a comprehensive cloud cost optimization dashboard for FinOps teams, focusing on advanced features and technical specifications

### Prompt

```
Design a comprehensive cloud cost optimization dashboard for FinOps teams, focusing on advanced features and technical specifications. Outline the key components, features, and best practices for effective cloud expenditure management and cost transparency. Include real-world examples of successful dashboard implementations, highlighting their benefits and metrics achieved.** Address the following aspects in your design:** ## 1. Data Integration and Visualization ## 2. Cost Allocation and Tagging ## 3. Budgeting and Forecasting ## 4. Alerting and Notification Systems ## 5. Security and Access Control ## 6. Scalability and Customization Ensure your design reflects the needs of advanced users and provides sophisticated solutions for cloud cost optimization.
```

### What it does

- Provides a structured framework for cloud architecture and cost optimization.
- Covers service selection, security, compliance, and disaster recovery planning.
- Helps evaluate cloud providers and migration strategies for your workload.

### Tips

- Specify your cloud provider (AWS / Azure / GCP) at the start for provider-specific guidance.
- Mention regulatory requirements (GDPR, HIPAA) so the response can address compliance directly.
- Ask for a cost estimate breakdown when evaluating architecture decisions.

### How to use

1. Open ChatGPT or Claude and paste the prompt.
2. Provide details about your current infrastructure, cloud provider preference, and compliance needs.
3. Work through the framework section by section, requesting elaboration on complex areas.
4. Use the output as the basis for a technical proposal or architecture review document.
