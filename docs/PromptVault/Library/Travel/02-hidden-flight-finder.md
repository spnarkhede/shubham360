### 2. Hidden Flight Finder

**Use Case:** Discover lower-cost routes including less obvious carriers and connections.

**Target Personas:** Budget Traveller, Frequent Traveller, Independent Traveller

**Tags:** `Travel`, `Planning`, `Cost Optimization`

**Prompt:**

```
Find all available flights from [origin] to [destination] over the next [X weeks], including budget airlines, regional carriers, and unconventional routes.
Rank options by total final cost, not only base fare.
```
