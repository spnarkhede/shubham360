### 9. Verified Promotions Detector

**Use Case:** Validate airline promotions against trusted sources before purchase.

**Target Personas:** Budget Traveller, Deal Hunter, Frequent Traveller

**Tags:** `Travel`, `Research`, `Verification`

**Prompt:**

```
Find active promotional codes and flash sales for [airlines].
Validate source credibility and expiration dates.
Return only verified offers and discard unconfirmed deals.
```
